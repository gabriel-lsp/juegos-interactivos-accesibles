# Condiciones de uso autorizado

## Juegos Interactivos Accesibles - JIA

Juegos Interactivos Accesibles, identificado con las siglas JIA, forma parte del Ecosistema Virtual Accesible, EVA. Es un proyecto personal desarrollado, organizado y documentado por el Psicólogo Gabriel Berrospi, con finalidad educativa, pedagógica, accesible, inclusiva y no comercial.

El uso de este módulo por parte de instituciones educativas, docentes, familias, profesionales, organizaciones o comunidad educativa no implica transferencia de autoría, titularidad ni cesión total de derechos sobre la estructura digital, código fuente, organización de contenidos, diseño funcional, lógica de funcionamiento, textos propios, presentación web, documentación ni sistematización desarrollada por el autor.

## Uso permitido

Se permite la consulta y difusión del enlace oficial del proyecto con fines educativos, pedagógicos, formativos, familiares, comunitarios, de sensibilización, accesibilidad e inclusión, siempre que no exista finalidad comercial directa o indirecta y se conserve el reconocimiento de autoría correspondiente.

El recurso puede emplearse como apoyo en sesiones pedagógicas, talleres, capacitaciones, actividades familiares, acciones de sensibilización y experiencias de aprendizaje vinculadas a la atención a la diversidad.

## Reconocimiento obligatorio
