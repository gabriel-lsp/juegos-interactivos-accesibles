# Bitácora de cambios

Este documento registra los avances principales del repositorio Juegos Interactivos Accesibles.

## Versión inicial

Se crea el repositorio como espacio digital personal para organizar actividades lúdicas e interactivas orientadas a la accesibilidad, la inclusión educativa y la atención a la diversidad.

Se incorpora el README y la carpeta `docs` para ordenar la autoría, fuentes, créditos, uso permitido, alcance pedagógico, estructura del proyecto, criterios pedagógicos, catálogo de juegos y guía de uso.

## Pendientes

- Incorporar actividades iniciales.
- Organizar recursos visuales y datos.
- Crear instrucciones breves para docentes y familias.
- Revisar accesibilidad visual y navegación.
