# Bitácora de cambios

Este documento registra los avances principales del repositorio Juegos Interactivos Accesibles.

La bitácora permite dejar evidencia del desarrollo progresivo del proyecto, facilitar la revisión institucional y diferenciar las versiones iniciales de futuras mejoras.

## Versión inicial

Se crea el repositorio Juegos Interactivos Accesibles como espacio digital para organizar actividades lúdicas e interactivas orientadas a la accesibilidad, la inclusión educativa y la atención a la diversidad.

Se incorpora un archivo `README.md` ampliado con finalidad del proyecto, público objetivo, alcance pedagógico, estructura sugerida, autoría, licencia y estado de avance.

Se agregan documentos de respaldo en la carpeta `docs` para sustentar el uso pedagógico, las fuentes, los créditos, el uso permitido, el respaldo institucional y la organización del proyecto.

## Pendientes de desarrollo

- Definir las primeras categorías de juegos.
- Elaborar o incorporar actividades interactivas iniciales.
- Organizar recursos visuales y datos del proyecto.
- Crear instrucciones breves para docentes y familias.
- Revisar accesibilidad visual, navegación y compatibilidad con dispositivos.
- Registrar fuentes externas utilizadas.
- Diferenciar versiones institucionales, personales o derivadas, si corresponde.

## Criterios para registrar cambios futuros

Cada mejora significativa debe anotarse con fecha, descripción breve y motivo del cambio.

Formato sugerido:

```text
Fecha: día/mes/año
Cambio realizado:
Motivo:
Responsable:
Observaciones:
```

Este registro no reemplaza los commits de GitHub, pero ayuda a documentar el sentido pedagógico e institucional de cada modificación importante.
