# Uso permitido

Este documento orienta el uso permitido del repositorio Juegos Interactivos Accesibles.

Juegos Interactivos Accesibles es un proyecto educativo digital personal desarrollado y organizado por Gabriel Berrospi. Tiene una finalidad educativa, accesible y no comercial.

## Usos permitidos

El proyecto puede utilizarse para:

- Actividades educativas con estudiantes.
- Apoyo a docentes y familias.
- Sensibilización sobre accesibilidad e inclusión educativa.
- Elaboración de recursos pedagógicos derivados, conservando los créditos correspondientes.
- Mejora o adaptación del código, respetando la licencia vigente.
- Uso como referencia para proyectos educativos similares.

## Condiciones de uso

Toda reutilización debe conservar:

- El nombre del proyecto.
- La autoría de Gabriel Berrospi.
- Los créditos de recursos externos utilizados.
- La diferenciación de cambios cuando se realicen adaptaciones.

## Recursos externos

El código del repositorio se rige por la licencia incluida en `LICENSE`.

Los materiales externos, imágenes, logos, audios, videos, textos o recursos de terceros mantienen sus propias condiciones de autoría, uso y difusión.

## Límites de uso

No se debe presentar el proyecto como propio, eliminar créditos o publicar versiones modificadas sin identificar claramente los cambios realizados.

Este repositorio debe entenderse como apoyo educativo digital. No sustituye la planificación docente, la evaluación profesional ni la intervención especializada.
